# dsa-cpp 🚀
**Zero-to-hero Data Structures & Algorithms in C++20**

> Every topic is implemented from scratch with full tests and demos,
> committed to this repository after each session.

## Build Requirements
| Tool | Version |
|------|---------|
| C++ Standard | C++20 |
| g++ (MSYS2 UCRT64) | ≥ 13.x |
| MSVC (VS 2022) | ≥ 19.3x |
| CMake | ≥ 3.25 |
| Ninja | any recent |

## Compile Flags
```
g++:  -std=c++20 -Wall -Wextra -O2
MSVC: /std:c++20 /W4 /O2
```
